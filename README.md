# Morphogrammer — перекодировщик похожих букв

Это простая веб‑страница, которая «переодевает» кириллический текст в визуально похожие буквы других алфавитов — латиницы, греческого и иврита.

## Что умеет страница

### Перекодирование текста

- Вы пишете текст в левое поле и нажимаете «Перекодировать».
- В правом поле появляется тот же текст, но буквы заменены на максимально похожие по начертанию:
  - всегда есть замены на латиницу;
  - при включении «Греческий» — там, где есть греческий аналог, используется он;
  - при включении «Иврит» — приоритет отдаётся ивриту, потом греческому и только затем латинице.

### Настройка соответствий букв

- Внизу есть таблица с одной строкой и четырьмя выпадающими списками:
  - 1‑й столбец — буква кириллицы;
  - 2‑й — её замена латиницей;
  - 3‑й — греческая буква;
  - 4‑й — буква иврита.
- Выбираете букву в первом столбце — остальные три автоматически показывают текущие замены для неё.
- Если в одном из правых столбцов выбрать другую букву, именно эта пара «кириллица → другая буква» будет использоваться при перекодировании.

### Переключатели языков

- «Греческий» — включить/выключить использование греческих букв (если для конкретной буквы они есть).
- «Иврит» — включить/выключить использование ивритских букв с наивысшим приоритетом.

### Дополнительные функции

- «Сброс» — вернуть всё к настройкам по умолчанию и очистить поля ввода/вывода.

### Особенность работы с ивритом

- Браузеры по‑умолчанию показывают иврит справа налево, из‑за чего группы ивритских букв могут «переворачиваться».
- Страница автоматически добавляет невидимые служебные символы только в тех местах, где подряд идёт две и более ивритских букв, чтобы сохранить привычный порядок отображения. Это не влияет на внешний вид текста, но помогает избежать визуальной «зеркальности».
